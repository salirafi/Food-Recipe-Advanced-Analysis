Dataset Overview
Recipe Dataset
Exploratory Analysis

This project explores a large-scale recipe dataset from multiple perspectives, ranging from ingredient relationships and nutritional composition to cooking complexity and user perception. Rather than following a single rigid narrative, the analysis is intentionally exploratory, with different patterns and insights that may emerge naturally across the data. The focus is on combining unique approach with visually engaging representations to create an experience that is both informative and enjoyable to navigate. Lastly, it does not serve as an academic report or anything similar, and is instead used simply for the author's learning experience and the reader's generic knowledge.
